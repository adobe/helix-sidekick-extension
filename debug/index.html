Helix Sidekick | Debug Server

  1. Install a Sidekick bookmarklet
  2. Right-click bookmarklet
  3. Click "Edit"
  4. Change URL to load 
    http://localhost:3001/src/sidekick/app.js
    instead of 
    https://wwww.hlx.live/tools/sidekick/app.js
  5. Click "Save"
  6. Navigate to a page of your Helix project and invoke Sidekick bookmarklet
