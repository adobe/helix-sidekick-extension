Sidekick | Debug Server

  1. Disable the Chrome extension if you have it installed
  2. Install a Sidekick bookmarklet
  3. Right-click bookmarklet
  4. Click "Edit"
  5. Change URL to load 
    http://localhost:3001/src/bookmarklet/app.js
    instead of 
    https://www.aem.live/tools/sidekick/app.js
  6. Click "Save"
  7. Navigate to a page on your site and invoke Sidekick bookmarklet
