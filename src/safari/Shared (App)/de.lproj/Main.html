HW Sidekick kann über „Einstellungen“ aktiviert/deaktiviert werden:

  1. Öffnen Sie die App „Einstellungen“.
  2. Tippen Sie auf „Safari“ und anschließend auf „Erweiterungen“.
  3. Aktivieren bzw. deaktivieren Sie HW Sidekick.

Sie können HW Sidekick in den Einstellungen für Safari-Erweiterungen aktivieren.

HW Sidekick ist zurzeit aktiviert. Sie können HW Sidekick in den Einstellungen für Safari-Erweiterungen deaktivieren.

HW Sidekick ist zurzeit deaktiviert. Sie können HW Sidekick in den Einstellungen für Safari-Erweiterungen aktivieren.
