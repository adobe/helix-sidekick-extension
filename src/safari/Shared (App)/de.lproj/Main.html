AEM Sidekick kann über „Einstellungen“ aktiviert/deaktiviert werden:

  1. Öffnen Sie die App „Einstellungen“.
  2. Tippen Sie auf „Safari“ und anschließend auf „Erweiterungen“.
  3. Aktivieren bzw. deaktivieren Sie AEM Sidekick.

Sie können AEM Sidekick in den Einstellungen für Safari-Erweiterungen aktivieren.

AEM Sidekick ist zurzeit aktiviert. Sie können AEM Sidekick in den Einstellungen für Safari-Erweiterungen deaktivieren.

AEM Sidekick ist zurzeit deaktiviert. Sie können AEM Sidekick in den Einstellungen für Safari-Erweiterungen aktivieren.
